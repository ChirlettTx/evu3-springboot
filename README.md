# Evaluación Unidad 3 – Spring Boot

Proyecto desarrollado en Spring Boot para la Evaluación de la Unidad 3.

## Tecnologías
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Estructura del proyecto
- model: entidades JPA
- repository: repositorios
- service: lógica de negocio
- controller: controladores REST

## Ejecución del proyecto
```bash
./mvnw spring-boot:run
